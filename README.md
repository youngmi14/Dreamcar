- 인원 : 프론트엔드 3명 & 백엔드 2명  
- Backend Repositoty 주소: https://github.com/wecode-bootcamp-korea/DreamCar-backend

### 목표

- 다양한 기능(스크롤 이벤트, color 바뀜, transition효과 등) 구현해보기
- 백엔드 API를 통해 원하는 데이터 가져오기
- SASS 활용해서 작업 해보기
- 리액트의 lifecycle 이해하여 이벤트 적용하기
